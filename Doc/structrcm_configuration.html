Detailed Description

This structure is used in rcmMsg_SetConfigRequest and rcmMsg_GetConfigConfirm to configure basic operation parameters of the radio.

This structure is needed to build the rcmMsg_SetConfigRequest and rcmMsg_GetConfigConfirm packet.

See also
rcmMsg_SetConfigRequest; rcmMsg_GetConfigConfirm

Member Data Documentation

◆ integrationIndex

rcrm_uint16_t rcmConfiguration::integrationIndex

Pulse integration Index(PII). Sets the pulses per data symbol of the radio. Values 4-9 are valid.

Note
Default value is 7. (2^7 = 128 pulses per data symbol)

◆ antennaMode

rcrm_uint8_t rcmConfiguration::antennaMode

Antenna Mode. Sets the antenna used for TX and RX.

• 0 = A[TX/RX]

• 1 = B[TX/RX]

• 2 = A[TX]B[RX]

• 3 = A[RX]B[TX])

In addition, setting the high order bit (0x80) of this byte enables automatic toggling of the antenna for each superframe (In RangeNet/Networking TDMA, even-numbered superframes use the “base” antenna mode, and odd-numbered superframes use the inverse antenna mode)

Note
Default value is 0(0x00)

◆ codeChannel

rcrm_uint8_t rcmConfiguration::codeChannel

Code Channel. Sets the active UWB channel. This allows multiple ranging conversations to occur simultaneously. The requester and responder must both be on the same channel for successful communication. The available channels are "long" codes (0-10) and "short" codes (16-75).

Note
Default value is 0

◆ electricalDelayPsA

rcrm_int32_t rcmConfiguration::electricalDelayPsA

Antenna A Delay. Sets the time delay (in picoseconds) of the SMA cable for Antenna A.

Note
Default value is 0ps. 0ps corresponds to a Broadspec antenna connected via a 90 degreee elbow

◆ electricalDelayPsB

Antenna B Delay. Sets the time delay (in picoseconds) of the SMA cable for Antenna B.

Note
Default value is 0ps. 0ps corresponds to a Broadspec antenna connected via a 90 degreee elbow

◆ txGain

rcrm_uint8_t rcmConfiguration::txGain

Transmit Gain. Specifies the active UWB transmit power from 0 (lowest) to 63 (highest). The actual transmit power depends on the radio model. See the model's data sheet for additional information.
