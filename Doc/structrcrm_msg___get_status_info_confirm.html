Detailed Description

Confirm message from the radio for GetStatusInfoLegacyRequest. Reports status info and version numbers.

Note
This is a legacy message, replaced by the more focused rcrmMsg_GetStatusInfoConfirm and rcrmMsg_GetVersionInfoConfirm. Although it will continue to be supported for the forseeable future as of April 2020, new software should use those messages.
