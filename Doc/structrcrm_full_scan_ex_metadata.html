◆ scanType

rcrm_uint8_t rcrmFullScanExMetadata::scanType

Scan type - range, monostatic, etc.

• 1 (RCRM_SCAN_TYPE_RANGE) - Scan came from a range request or response

• 2 (RCRM_SCAN_TYPE_BISTATIC) - Scan came from a non-range packet received from another radio

• 3 (RCRM_SCAN_TYPE_MONOSTATIC) - Scan came from this radio simultaneously transmitting and receiving (monostatic radar)

• 4 (RCRM_SCAN_TYPE_AMBIENT) - Scan came from this radio sampling the RF environment

• 5 (RCRM_SCAN_TYPE_ACQ_STREAM_NOISE) - Background noise debug data streamed from acquisition module

• 6 (RCRM_SCAN_TYPE_ACQ_STREAM_THRESH) - Acq threshold debug data streamed from acquisition module

• 7 (RCRM_SCAN_TYPE_ACQ_RANGE_NOISE) - (Not currently implemented) Background noise debug data from acquisition module, captured while waiting for a range response.

• 8 (RCRM_SCAN_TYPE_ACQ_RANGE_THRESH) - (Not currently implemented) Acq threshold debug data from acquisition module, captured while waiting for a range response.

rcrm_uint8_t rcrmFullScanExMetadata::scanCompressionType

Scan compression type

0

RCRM_SCAN_COMPRESSION_NONE

Uncompressed scan data (default).

2

RCRM_SCAN_COMPRESSION_INT16

Upper two bytes are removed from each sample to create 16-bit signed integer samples. If the scan integration is high enough to potentially exceed 16 bits in scan samples, the scan samples are downscaled to fit.

◆ noise

rcrm_uint16_t rcrmFullScanExMetadata::noise

These flags indicate characteristics of the scan: • 1 (Bit 0) = SATURATED

• 2 (Bit 1) = SCAN WINDOW TOO SHORT

• 4 (Bit 2) = SNR TOO LOW

• 8 (Bit 3) = LINE OF SIGHT (LOS)

• 16 (Bit 4) = NON-LINE OF SIGHT (NLOS)

• 32 (Bit 5) = LINE-OF-SIGHT PEAK MISMATCH

• 128 (Bit 7) = RMS Delay Spread Squared and Noise Ratio Thresholds Exceeded (internal LED metric)

• 256 (Bit 8) = Norm Squared Residual Threshold Exceeded (internal LED metric) Noise represents the noise generated in the receiver, estimated from the scan. Along with Vpeak, Noise can be used to compute an SNR estimate using 20 * Log10 (Vpeak / Noise).
