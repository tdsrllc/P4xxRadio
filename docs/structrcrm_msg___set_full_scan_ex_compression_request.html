◆ scanCompressionType

rcrm_uint8_t rcrmMsg_SetFullScanExCompressionRequest::scanCompressionType

Scan data compression selection. Selections other than the default of no compression would typically only be used on slower/low bandwidth interfaces.

0RCRM_SCAN_COMPRESSION_NONEUncompressed scan data (default).
2RCRM_SCAN_COMPRESSION_INT16Upper two bytes are removed from each sample to create 16-bit signed integer samples. If the scan integration is high enough to potentially exceed 16 bits in scan samples, the scan samples are downscaled to fit.

◆ persistFlag

rcrm_uint8_t rcrmMsg_SetFullScanExCompressionRequest::persistFlag

Persist Flag. Sets whether the Config Request is stored in Flash and persist through power cycling. Default value is 0

Note
• 0 does not write anything to flash.

• 1 will save all active radio settings to flash.

• 2 will save only the settings updated by this message to flash.

Warning
ALL radio settings set after the persist flag is set to 1 will also be saved to flash.
