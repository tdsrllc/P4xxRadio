Detailed Description

Full Neighbor Data Base message structure.

Note
This message may also be sent automatically (with message type RN_FULL_NEIGHBOR_DATABASE_INFO) if the Autosend flags and interval are appropriately configured in the RangeNet configuration. If autosend is enabled, the message type for the FullNeighborDatabaseInfo message is (0x3203)

◆ antennaMode

rcrm_uint8_t rnFullNeighborDatabaseEntry_t::antennaMode

Antenna Mode. Sets the antenna used for TX and RX.

• 0 = A[TX/RX]

• 1 = B[TX/RX]

• 2 = A[TX]B[RX]

• 3 = A[RX]B[TX])

In addition, setting the high order bit (0x80) of this byte enables automatic toggling of the antenna for each superframe (even-numbered superframes use the “base” antenna mode, and odd-numbered superframes use the inverse antenna mode)

Note
Default value is 0(0x00)

◆ flags

rcrm_uint8_t rnFullNeighborDatabaseEntry_t::flags

Bit flags for various parameters.

• Bit 0 – Beacon mode: 0 = Normal mode, 1 = Beacon mode

• Bit 1 – Do not range to me: 0 = Normal, 1 = Do not range to me

• Bit 2 – In Exclusion List: 0 = Not excluded, 1 = Excluded

• Bit 3 – Range Uncalibrated: 0 = Normal, 1 = Uncalibrated

• Bits 4:7 – Reserved
