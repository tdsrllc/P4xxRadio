Detailed Description

Request message from the host that configures the radio's exlusion list.

The exclusion list is a user defined list of Node IDs that the radio will not range to and exclude from its Neighbor Database.

Note
This is a variable length packet dependent on the number of nodes in the exclusion list. The maximum of number of nodes that can be in the exclusion list is 256.
