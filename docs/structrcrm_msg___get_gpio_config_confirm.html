◆ gpioMode

rcrm_uint32_t rcrmMsg_GetGpioConfigConfirm::gpioMode

Specifies the function for each of the 15 GPIOs, 2 bits per GPIO. The function for GPIO 0 is mapped to GPIO Mode bits [1:0]. GPIO 1 is mapped to bits [3:2], and so on. A function of 00b selects Function 0 (general purpose I/O), while other functions are specific to the GPIO. The GPIOs and functions are listed below. Default settings are in bold.

◆ gpioDirection

rcrm_uint16_t rcrmMsg_GetGpioConfigConfirm::gpioDirection

Sets the direction of the GPIO (input or output). Each GPIO maps to 1 bit (GPIO 0 maps to bit 0, GPIO 1 to bit 1, etc.). Only valid for a GPIO if that GPIO's function is set to Function 0. Set the GPIO's corresponding bit to 0 to operate as an input, or set it to 1 to operate as an output.
