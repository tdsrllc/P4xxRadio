Detailed Description

Host request message that commands the Radio to send a UWB data-only packet(without range request)

Note
All radios within range will receive this data and send to their respective Host(if attached). No automatic acknowledgement is provided for data.
See also
Corresponding Message: rcmMsg_SendDataConfirm

◆ antennaMode

rcrm_uint8_t rcmMsg_SendDataRequest::antennaMode

Antenna Mode. Sets the antenna used for TX and RX.

• 0 = A[TX/RX]

• 1 = B[TX/RX]

• 2 = A[TX]B[RX]

• 3 = A[RX]B[TX])

In addition, setting the high order bit (0x80) of this byte enables automatic toggling of the antenna for each superframe (even-numbered superframes use the “base” antenna mode, and odd-numbered superframes use the inverse antenna mode)

Note
Default value is 0(0x00)
