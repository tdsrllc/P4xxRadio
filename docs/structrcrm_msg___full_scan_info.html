Detailed Description

Optional waveform scan of the RF pulse sent from radio whenever it receives a UWB packet that supports scanning (e.g. ranging packets)

This is debug information only and is not sent by default. The default state is defined by the rcmMsg_SetConfigRequest message.

See also
rcmMsg_SetConfigRequest
Note
The scan info messages are large and have to be broken up in multiple messages.

◆ scanType

rcrm_uint8_t rcrmMsg_FullScanInfo::scanType

Scan type - range, monostatic, etc.

• 1 (RCRM_SCAN_TYPE_RANGE) - Scan came from a range request or response

• 2 (RCRM_SCAN_TYPE_BISTATIC) - Scan came from a non-range packet received from another radio

• 3 (RCRM_SCAN_TYPE_MONOSTATIC) - Scan came from this radio simultaneously transmitting and receiving (monostatic radar)

• 4 (RCRM_SCAN_TYPE_AMBIENT) - Scan came from this radio sampling the RF environment
