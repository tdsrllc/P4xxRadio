◆ noise

rcrm_uint16_t rcrmMsg_ScanInfo::noise

Noise represents the noise generated in the receiver from the received waveform scan. Along with Vpeak, Noise can be used to compute SNR using 20 * Log10 (Vpeak / Noise). Due to processing constraints in the radio CPU, this noise is an estimate based on a partial rather than full analysis of the waveform scan.
