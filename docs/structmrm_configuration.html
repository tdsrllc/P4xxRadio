rcrm_uint16_t mrmConfiguration::baseIntegrationIndex

Minimum integration for the scan, as a power of 2 (integration 10 = 2^10 = 1024 pulses per scan sample)

Note
Must be in the range of 4 to 15, unless segmentation is used, in which case this must be in the range of 4 to 10

◆ segmentIntegration

rcrm_uint8_t mrmConfiguration::segmentIntegration[MRM_MAX_SEGMENT_COUNT]

Additional integration on top of the base integration. Due to the way the system works, if segmentation is used, the Base Integration Index must be in the range of 4 to 10, and Segment Integrations, while independent of each other, must be in the range 0 to 5.

Note
Total integration for a segment is (baseIntegrationIndex + the corresponding segmentIntegration).

◆ antennaMode

rcrm_uint8_t mrmConfiguration::antennaMode

Antenna Mode. Sets the antenna used for TX and RX.

• 0 = A[TX/RX]

• 1 = B[TX/RX]

• 2 = A[TX]B[RX]

• 3 = A[RX]B[TX])

In addition, setting the high order bit (0x80) of this byte enables automatic toggling of the antenna for each superframe (In RangeNet/Networking TDMA, even-numbered superframes use the “base” antenna mode, and odd-numbered superframes use the inverse antenna mode)

Note
Default value is 0(0x00)

◆ txGain

rcrm_uint8_t mrmConfiguration::txGain

Transmit Gain. Specifies the active UWB transmit power from 0 (lowest) to 63 (highest). The actual transmit power depends on the radio model. See the model's data sheet for additional information.

◆ codeChannel

rcrm_uint8_t mrmConfiguration::codeChannel

Code Channel. Sets the active UWB channel. This allows multiple ranging conversations to occur simultaneously. The requester and responder must both be on the same channel for successful communication. The available channels are "long" codes (0-10) and "short" codes (16-75).

Note
The default value is 0

◆ persistFlag

rcrm_uint8_t mrmConfiguration::persistFlag

Persist Flag. Sets whether the Config Request is stored in Flash and persist through power cycling. Default value is 0

Note
• 0 does not write anything to flash.

• 1 will save all active radio settings to flash.

• 2 will save only the settings updated by this message to flash.

Warning
ALL radio settings set after the persist flag is set to 1 will also be saved to flash.
