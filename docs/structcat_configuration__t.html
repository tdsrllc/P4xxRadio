◆ antennaMode

rcrm_uint8_t catConfiguration_t::antennaMode

Antenna Mode. Sets the antenna used for TX and RX.

• 0 = A[TX/RX]

• 1 = B[TX/RX]

• 2 = A[TX]B[RX]

• 3 = A[RX]B[TX])

In addition, setting the high order bit (0x80) of this byte enables automatic toggling of the antenna for each superframe (In RangeNet/Networking TDMA, even-numbered superframes use the “base” antenna mode, and odd-numbered superframes use the inverse antenna mode)

Note
Default value is 0(0x00)

◆ codeChannel

rcrm_uint8_t catConfiguration_t::codeChannel

Code Channel. Sets the active UWB channel. This allows multiple ranging conversations to occur simultaneously. The requester and responder must both be on the same channel for successful communication. The available channels are "long" codes (0-10) and "short" codes (16-75).

Note
Default value is 0

◆ persistFlag

rcrm_uint8_t catConfiguration_t::persistFlag

Persist Flag. Sets whether the Config Request is stored in Flash and persist through power cycling. Default value is 0

Note
• 0 does not write anything to flash.

• 1 will save all active radio settings to flash.

• 2 will save only the settings updated by this message to flash.

Warning
ALL radio settings set after the persist flag is set to 1 will also be saved to flash.
