Detailed Description

Request message from the host to set the data buffer in the RangeNet range request packet.

This data will be transmitted by the unit whenever it sends a range request packet.
The buffer will remain in effect until changed by the Host. Upon boot this buffer will be empty. If the amount of data in the buffer is greater than the Max Request Data Size value in the RangeNet configuration, the buffer will be truncated.
